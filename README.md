# Medical Data Analysis & Prediction

Проект по анализу медицинских данных и прогнозированию заболеваний и стоимости лечения.

## Описание проекта

Анализ синтетического датасета с медицинскими данными пациентов, включающий:
Очистку и предобработку данных
Разведочный анализ (EDA)
Прогнозирование заболеваний (Medical Condition)
Прогнозирование стоимости лечения (Billing Amount)

## Задачи проекта

### 1. Подготовка данных
Обработка пропущенных значений
Удаление дубликатов
Feature engineering

### 2. Разведочный анализ (EDA)
Анализ распределений и корреляций
Проверка гипотез
Визуализация ключевых метрик

### 3. Классификация - предсказание заболевания
Модель: Stacking Classifier (Random Forest + XGBoost + SVM)
Метрика: Accuracy

### 4. Регрессия - предсказание стоимости лечения
Модель: Random Forest Regressor
Метрики: R², RMSE, MAPE


## Технологии

- **Python**
- **Pandas, NumPy** - обработка данных
- **Scikit-learn** - ML модели
- **XGBoost** - градиентный бустинг
- **Matplotlib, Seaborn** - визуализация
- **Gender-guesser** - определение пола по имени
